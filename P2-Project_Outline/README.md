## Submission Instructions

### Overview
Our website aims to provide information about local camps and extracurricular activities for students in grades K-12. The platform will allow parents and guardians to search for activities based on location, age group, and interests. Our goal is to make it easy for families to find and choose the right programs for their children, ultimately promoting a love for learning and personal growth.

### Features

Authentication: Secure user login 

Search: Users can search for camps and activities based on location, age group, and interests.

Reviews and Ratings: Users can leave reviews and ratings for camps and activities they have participated in, providing valuable feedback to other parents and guardians.

Favorites: Users can add camps and activities to a personalized favorites list, making it easy to keep track of interests.


### Technologies
HTML, CSS, and JavaScript for front-end development

Java for backend development

MySQL for database management

### What I'll Have to Learn
To make the search feature more efficient, we will need to integrate Angular with Java API to allow persistent data storage. We will also need to learn how to implement secure user authentication and authorization to protect users' personal information.
